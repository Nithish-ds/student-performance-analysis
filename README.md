# Student Performance Analysis

## About

This project analyzes the performance of 100 BCA Data Science students using Python, Pandas, and Matplotlib.

## Tools Used

- Python
- Pandas
- Matplotlib

## Dataset

The dataset contains:

- Student ID
- Name
- Age
- Department
- Semester
- CGPA
- Attendance

## Analysis Performed

- Missing Values Check
- Duplicate Records Check
- Average CGPA
- Highest CGPA Student
- Lowest CGPA Student
- Semester Wise Student Count
- Semester Wise Average CGPA
- Attendance Wise Average CGPA
- CGPA Statistics
